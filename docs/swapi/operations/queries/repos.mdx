List all bundle repositories with pagination, sorting, and search capabilities.

Returns a paginated list of repositories (bundles) that the authenticated user has access to
within the specified organization. Each repository can contain multiple releases with
different version strategies (stable vs development).


```graphql
repos(
  organizationId: ID!
  sort: ReposSort
  cursor: Cursor
  search: String
): ReposPage
```

#### [<code style={{ fontWeight: 'normal' }}>repos.<b>search</b></code>](#)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>String</b></code>](/swapi/types/scalars/string.mdx) <Badge class="badge badge--secondary " text="scalar"/> 
Full-text search with Google-style operators (AND, OR, -, quotes).
Searches across the most recent release's name, description, and operator guide.

Syntax:
- "word1 word2" → OR (matches either)
- "word1 AND word2" → both required
- "-word" → exclude term
- "\"exact phrase\"" → exact match

Examples:
- "postgres AND aws -aurora"
- "kubernetes AND (gcp OR azure)"
- "\"load balancer\""
