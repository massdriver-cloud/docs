Copy package params from one package to another with optional overrides.

Both packages must be from the same manifest (e.g., api-prod-database and api-staging-database).
Fields marked with `$md.copyable: false` in the params schema will be excluded from copying.
The merged params are deep merged with the overrides, allowing you to override specific values.

For complete documentation on the `$md.copyable` annotation and usage examples, see the [Massdriver Annotations Reference](/json-schema-cheat-sheet/massdriver-annotations#mdcopyable).


```graphql
copyPackageParams(
  organizationId: ID!
  srcPackageId: ID!
  destPackageId: ID!
  overrides: JSON
): PackagePayload
```
