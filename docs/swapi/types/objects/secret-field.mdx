Application secret definitions. These fields are defined in your applications massdriver.yaml file.

Secrets are only applied to `application` type bundles.


```graphql
type SecretField {
  name: String!
  required: Boolean!
  json: Boolean!
  title: String
  description: String
  valueMetadata: SecretMetadata
}
```
