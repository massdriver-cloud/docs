Statistics and metrics including bundle usage, deployment history, and active alarms.

type DashboardStatistics {
  bundleStatistics: [BundleStatistic]
  environmentStatistics: [EnvironmentStatistic]

  recentInfrastructureDeployments: [RecentDeployment]
  recentApplicationDeployments: [RecentDeployment]
}

### Fields

#### [<code style={{ fontWeight: 'normal' }}>DashboardStatistics.<b>bundleStatistics</b></code>](#)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>[BundleStatistic]</b></code>](/swapi/types/objects/bundle-statistic.mdx) <Badge class="badge badge--secondary " text="list"/> <Badge class="badge badge--secondary " text="object"/> 
Returns usage and cost metrics for each infrastructure module (bundle) across your organization. For each bundle, it provides total number of environments using the bundle, total number of package instances deployed, and total accumulated costs since inception.


#### [<code style={{ fontWeight: 'normal' }}>DashboardStatistics.<b>environmentStatistics</b></code>](#)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>[EnvironmentStatistic]</b></code>](/swapi/types/objects/environment-statistic.mdx) <Badge class="badge badge--secondary " text="list"/> <Badge class="badge badge--secondary " text="object"/> 
Shows cost trends for each environment by comparing two 30-day periods. The current period (last 30 days) is compared to the previous period (days 31-60) to calculate cost changes. A positive growth rate indicates rising costs, while a negative rate shows cost reduction.


#### [<code style={{ fontWeight: 'normal' }}>DashboardStatistics.<b>deploymentStatistics</b></code>](#)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>[DeploymentStatistic]</b></code>](/swapi/types/objects/deployment-statistic.mdx) <Badge class="badge badge--secondary " text="list"/> <Badge class="badge badge--secondary " text="object"/>
