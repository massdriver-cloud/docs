The decision returned by an `evaluatePolicy` request.

`action` and `entityId` mirror the inputs so batch callers can correlate
decisions with their original questions without tracking positions
externally.


```graphql
type PolicyDecision {
  allowed: Boolean!
  action: String!
  entityId: ID!
}
```
