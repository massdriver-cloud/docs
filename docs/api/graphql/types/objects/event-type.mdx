A single event type in the Massdriver event catalog.

The `id` field follows dotted `entity.action` notation in past tense
(e.g., `project.cloned`, `deployment.completed`).


```graphql
type EventType {
  id: ID!
  description: String!
  entity: String!
  action: String!
  changeType: EventAction!
}
```
