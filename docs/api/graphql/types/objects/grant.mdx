A grant: a specific OCI repo or resource shared with recipient projects /
environments matching `recipientConditions`.


```graphql
type Grant {
  id: ID!
  action: String!
  recipientConditions: Conditions!
  source: GrantSource!
  createdAt: DateTime!
  updatedAt: DateTime!
}
```
