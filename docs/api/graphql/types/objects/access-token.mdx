An access token's metadata, without the raw token value.

Use this type to list, inspect, and manage existing tokens. The full token value is only
available at creation time — if you need to identify a specific token, use its `prefix`
(e.g., `md_a1b2c3d4`).

**Token states:**
- **Active** — `revokedAt` is `null` and `expiresAt` is in the future.
- **Expired** — `expiresAt` is in the past. The token no longer works.
- **Revoked** — `revokedAt` is set. The token was manually disabled and no longer works.


```graphql
type AccessToken {
  id: ID!
  name: String!
  prefix: String!
  scopes: [String!]!
  expiresAt: DateTime!
  revokedAt: DateTime
  lastUsedAt: DateTime
  createdAt: DateTime!
  organization: Organization
}
```

The organization this token is scoped to, or `null` if the token has been revoked.

Hiding the organization on revoked tokens prevents an ex-member of the org from walking
`token.organization { name, logo, ... }` after their tokens have been revoked on removal.
The token row stays visible so the owner can audit revocation history; only the
organization edge is severed.
