A record of a sign-in to your Massdriver account.

Created each time you authenticate through a linked OAuth or OIDC identity. Use this list
in the security center to spot unfamiliar IPs or user agents and rotate credentials if
anything looks wrong.


```graphql
type LoginSessionViewer {
  id: ID!
  ipAddress: String!
  userAgent: String
  provider: String!
  createdAt: DateTime!
}
```
