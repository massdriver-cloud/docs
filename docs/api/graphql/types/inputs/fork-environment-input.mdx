Create or update a fork of an existing environment.

```graphql
input ForkEnvironmentInput {
  attributes: Map
  copyEnvironmentDefaults: Boolean
  copyRemoteReferences: Boolean
  copySecrets: Boolean
  decommissionProtection: Boolean
  description: String
  id: String!
  name: String!
}
```
