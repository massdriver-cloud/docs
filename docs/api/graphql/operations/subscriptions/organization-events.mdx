Subscribe to organization-level events via WebSocket.

Receives events for projects created in the organization, OCI repositories
created in the bundle catalog, and bundle versions published to those
repositories. Requires organization membership.

```graphql
subscription {
  organizationEvents(organizationId: "my-org") {
    ... on Event { action timestamp }
    ... on ProjectEvent {
      project { id name }
    }
    ... on OciRepoEvent {
      ociRepo { id name }
    }
    ... on BundleEvent {
      bundle { id name version }
    }
  }
}
```


```graphql
organizationEvents(
  organizationId: ID!
): OrganizationEventsPayload
```
