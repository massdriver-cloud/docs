The billing status of an organization's subscription.

Use this to display billing warnings, restrict actions for unpaid accounts, or guide
users toward resolving payment issues.


```graphql
enum ViewerBillingStatus {
  TRIAL
  ACTIVE
  PAST_DUE
  EXPIRED
  CANCELED
  SUSPENDED
  PAYMENT_PENDING
  PAYMENT_FAILED
}
```




### Values

#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>TRIAL</b></code>](#trial) \{#trial\} 
Free trial period. The organization has full access until the trial expires.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>ACTIVE</b></code>](#active) \{#active\} 
Subscription is active and in good standing.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>PAST&#x005F;DUE</b></code>](#past-due) \{#past-due\} 
A payment is overdue. Access continues but action is needed to avoid suspension.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>EXPIRED</b></code>](#expired) \{#expired\} 
The free trial has ended without a subscription. Functionality is limited.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>CANCELED</b></code>](#canceled) \{#canceled\} 
The subscription was canceled. The organization is in a grace period.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>SUSPENDED</b></code>](#suspended) \{#suspended\} 
The account is suspended due to billing issues. Most operations are blocked.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>PAYMENT&#x005F;PENDING</b></code>](#payment-pending) \{#payment-pending\} 
A payment is being processed. This is typically a brief transitional state.


#### [<code style={{ fontWeight: 'normal' }}>ViewerBillingStatus.<b>PAYMENT&#x005F;FAILED</b></code>](#payment-failed) \{#payment-failed\} 
The most recent payment attempt failed. Update payment details to restore access.
