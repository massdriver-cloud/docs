The type of lifecycle change that triggered an event.

Every event carries an action indicating whether the resource was created,
modified, or removed.


```graphql
enum EventAction {
  CREATED
  UPDATED
  DELETED
}
```




### Values

#### [<code style={{ fontWeight: 'normal' }}>EventAction.<b>CREATED</b></code>](#created) \{#created\} 
A new resource was created.


#### [<code style={{ fontWeight: 'normal' }}>EventAction.<b>UPDATED</b></code>](#updated) \{#updated\} 
An existing resource was modified (configuration, status, or metadata changed).


#### [<code style={{ fontWeight: 'normal' }}>EventAction.<b>DELETED</b></code>](#deleted) \{#deleted\} 
A resource was permanently removed.
