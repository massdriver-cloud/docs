#### [<code style={{ fontWeight: 'normal' }}>OciRepoReleaseChannelsSortField.<b>NAME</b></code>](#name) \{#name\} 
Sort by channel name. Channels are ordered: `latest` first, then `~major` channels numerically, then `~major.minor` channels numerically. Dev (`+dev`) variants sort after their stable counterpart.
