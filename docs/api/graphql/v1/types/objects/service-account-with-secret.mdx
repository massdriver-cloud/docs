A service account with its secret token included.

This type is only returned by the `createServiceAccount` mutation. The `secret` field is
the credential used for API authentication and **cannot be retrieved again** after creation.
Store it securely (e.g., in a secrets manager or CI/CD variable) before navigating away.


```graphql
type ServiceAccountWithSecret {
  id: ID!
  name: String!
  description: String
  secret: String!
  createdAt: DateTime!
  updatedAt: DateTime!
}
```
