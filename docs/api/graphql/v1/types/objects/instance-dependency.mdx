An input dependency consumed by an instance, keyed by the field handle that receives it.

Dependencies are resources from other instances wired in via connections. For
example, an application instance might have a `database` dependency that
receives the connection string from a database instance.


```graphql
type InstanceDependency {
  field: String!
  resource: Resource!
}
```
