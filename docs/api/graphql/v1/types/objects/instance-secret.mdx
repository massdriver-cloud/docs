Metadata about an encrypted secret attached to an instance.

Secrets are encrypted key-value pairs injected at deploy time. The API
never returns secret values -- only the name and timestamps are exposed.


```graphql
type InstanceSecret {
  name: String!
  createdAt: DateTime!
  updatedAt: DateTime!
}
```
