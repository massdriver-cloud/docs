A profile avatar image.

Avatars are uploaded via the `setAccountAvatar` mutation and served at a stable URL.
Supported formats: PNG, JPG, GIF, and WebP (max 1 MB).


```graphql
type AvatarViewer {
  id: ID!
  url: String!
  contentType: String!
  fileSize: Int!
  width: Int
  height: Int
  createdAt: DateTime!
  updatedAt: DateTime!
}
```
