A project's infrastructure blueprint.

The blueprint contains all components (bundle instances) and links (connections
between them) that define how your infrastructure fits together.


```graphql
type Blueprint {
  components(
    filter: ComponentsFilter
    sort: ComponentsSort
    cursor: Cursor
  ): ComponentsPage
  links(
    filter: LinksFilter
    sort: LinksSort
    cursor: Cursor
  ): LinksPage
}
```
