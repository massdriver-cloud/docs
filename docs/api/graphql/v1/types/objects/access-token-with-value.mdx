An access token with the raw token value included.

This type is only returned by the `createAccessToken` mutation. The `token` field contains
the full credential needed for API authentication and &#x002A;&#x002A;cannot be retrieved again&#x002A;&#x002A; after
this response. Store it securely before navigating away.


```graphql
type AccessTokenWithValue {
  id: ID!
  name: String!
  token: String!
  prefix: String!
  scopes: [String!]!
  expiresAt: DateTime!
  createdAt: DateTime!
}
```
