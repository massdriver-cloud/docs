Filter by deployment status.

All operators within a single filter are combined with **AND**. Use `in` to match
deployments in any of several statuses.

```graphql
{ "status": { "in": ["COMPLETED", "FAILED"] } }
```


```graphql
input DeploymentStatusFilter {
  eq: DeploymentStatus
  in: [DeploymentStatus!]
}
```
