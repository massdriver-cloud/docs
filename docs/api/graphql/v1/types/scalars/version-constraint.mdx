Version constraint for bundle releases.

Accepts exact versions or release channels:
- `1.2.3` — exact version
- `1.2.3-dev.20060102T150405Z` — exact dev release
- `~1.2` — latest patch in 1.2.x series
- `~1` — latest minor in 1.x.x series
- `~1.2+dev` — latest in 1.2.x including dev builds
- `latest` — newest stable release
- `latest+dev` — newest release including dev builds
