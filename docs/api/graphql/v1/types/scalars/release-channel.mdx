A release channel identifier for automatic version tracking.

Release channels are version constraints that always resolve to the latest
matching published version. When you pin a component to a release channel,
it automatically picks up new compatible versions as they are published.

&#x002A;&#x002A;Stable channels&#x002A;&#x002A; (exclude development builds):

| Channel | Matches |
|---------|---------|
| `latest` | Newest stable release across all major versions |
| `~1` | Latest minor + patch in the `1.x.x` series |
| `~1.2` | Latest patch in the `1.2.x` series |

&#x002A;&#x002A;Development channels&#x002A;&#x002A; (include `-dev.*` pre-release builds):

| Channel | Matches |
|---------|---------|
| `latest+dev` | Newest release of any kind |
| `~1+dev` | Latest in `1.x.x` including dev builds |
| `~1.2+dev` | Latest in `1.2.x` including dev builds |
