Bundle identifier in `name@version` format.

Returns the fully resolved version of a bundle (e.g., `aws-aurora-postgres@1.2.3`
or `aws-aurora-postgres@1.2.3-dev.20060102T150405Z`).

As an input, accepts:
- `aws-aurora-postgres` — bare name, resolves to latest
- `aws-aurora-postgres@1.2.3` — exact version
- `aws-aurora-postgres@~1.2` — release channel (resolved server-side)
- `aws-aurora-postgres@latest` — latest stable release
- `aws-aurora-postgres@latest+dev` — latest including dev builds
