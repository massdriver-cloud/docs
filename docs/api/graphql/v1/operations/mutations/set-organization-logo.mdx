Upload or replace your organization's logo.

Accepts PNG, JPG, GIF, or WebP images up to 1 MB. If a logo already exists,
it is replaced. Use `removeOrganizationLogo` to delete without replacement.


```graphql
setOrganizationLogo(
  organizationId: ID!
  logo: Upload!
): LogoOrganizationPayload
```
