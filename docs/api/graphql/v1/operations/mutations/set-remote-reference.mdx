Set a remote reference on an instance, linking one of its resource fields to an
external resource.

The target resource can be from another project or an imported resource. The instance
must **not** be in `PROVISIONED` or `FAILED` status — you can only set remote references
on instances that haven't been deployed yet or have been decommissioned.

If this is the first remote reference on the instance, the instance status transitions
to `EXTERNAL`.

```graphql
mutation {
  setRemoteReference(
    organizationId: "my-org"
    input: {
      instanceId: "my-app"
      field: "aws_authentication"
      resourceId: "shared-creds-abc123"
    }
  ) {
    result { id field resource { id name } }
    successful
    messages { field message }
  }
}
```
