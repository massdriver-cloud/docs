Remove a remote reference from an instance.

The reference can only be removed if no provisioned instances are connected
through it. Removing the last remote reference resets the instance status
from EXTERNAL back to INITIALIZED.




```graphql
removeRemoteReference(
  organizationId: ID!
  input: RemoveRemoteReferenceInput!
): RemoteReferencePayload
```
