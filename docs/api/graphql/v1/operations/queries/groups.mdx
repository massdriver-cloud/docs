List all groups in your organization.

Returns a paginated, sortable list of groups. By default, groups are sorted
alphabetically by name.

**Example:**

```graphql
query {
  groups(organizationId: "my-org") {
    data {
      id
      name
      role
      description
    }
    cursor { next }
  }
}
```


```graphql
groups(
  organizationId: ID!
  sort: GroupsSort
  cursor: Cursor
): GroupsPage
```
