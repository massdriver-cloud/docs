List audit log events for your organization.

Returns a paginated, filterable, sortable list of audit log events. By default, events are
sorted newest first (`occurredAt` descending).

&#x002A;&#x002A;Filtering&#x002A;&#x002A; — Use the `filter` argument to narrow results by time range, event type,
actor type, or specific actor. All filters are combined with AND logic.

&#x002A;&#x002A;Example:&#x002A;&#x002A;

```graphql
query {
  auditLogs(
    organizationId: "my-org"
    filter: {
      occurredAt: { after: "2025-01-01T00:00:00Z" }
      type: { startsWith: "project." }
      actorType: { in: [ACCOUNT, SERVICE_ACCOUNT] }
    }
  ) {
    data {
      id
      occurredAt
      type
      subject
      actor { type name }
    }
    cursor { next }
  }
}
```


```graphql
auditLogs(
  organizationId: ID!
  filter: AuditLogsFilter
  sort: AuditLogsSort
  cursor: Cursor
): AuditLogsPage
```
