Subscribe to organization-level events via WebSocket.

Receives events for all projects and environments within the organization, including
project creation/deletion and environment lifecycle changes. Requires organization
membership.

```graphql
subscription {
  organizationEvents(organizationId: "my-org") {
    action
    timestamp
    ... on ProjectEvent {
      project { id name }
    }
    ... on EnvironmentEvent {
      environment { id name }
    }
  }
}
```


```graphql
organizationEvents(
  organizationId: ID!
): Event
```

### Type

#### [<code style={{ fontWeight: 'normal' }}><b>Event</b></code>](/api/graphql/v1/types/interfaces/event.mdx) <Badge class="badge badge--secondary " text="interface"/> 
Base interface implemented by all lifecycle events.

Every event carries an `action` describing what happened and a `timestamp` recording
when it occurred. Concrete event types add a field for the affected resource.

Use inline fragments to access the resource-specific fields:

```graphql
subscription {
  projectEvents(organizationId: "my-org", projectId: "my-project") {
    action
    timestamp
    ... on ComponentEvent {
      component { id name }
    }
    ... on LinkEvent {
      link { id fromField toField }
    }
  }
}
```
